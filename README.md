### 双指针法 （2-Remove_Element.cpp）
    -- 1.双指针法（快慢指针法）： 通过一个快指针和慢指针在一个for循环下完成两个for循环的工作,
        在数组和链表的操作中非常常见，很多数组、链表、字符串等操作，都使用双指针法。
    
    -- 2.判断指针移动的条件是双指针的核心：
        -- 移除元素：快慢指针都从数组的头开始
        -- 盛最多的水：快慢指针分别从数组的尾部和头部开始
